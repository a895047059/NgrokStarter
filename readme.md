# 内网穿透工具 NgrokStarter

## 必要条件
- ngrok官网申请免费账号和免费域名
- config.ini 填入
    ```
    [default]
    domain= 你的免费域名
    port = 你要映射本地的端口
    ```
- ngrok.yml 填入
    ```
    authtoken: 2UhPbl***Ye4dFci
    ```

## 如何运行
- python NgrokStarter.py
- NgrokStarter.exe

windows下运行软件，输入端口号或直接点击启动，可以启动该程序  
程序在几秒钟后会生成你域名的访问地址，可以直接复制到剪贴板访问
